Imagine que você queira desenvolver uma ferramenta para fazer algum cálculo muito complexo, um cálculo de valuation de uma empresa, analisar um portfólio de ações para tomar uma decisão de investimento, conectar sistemas que o N8N ainda não listou ou trabalhar com grandes conjuntos de dados para passar para o seu modelo. Enfim, não há limites. 

Número 2: Interface. Você pode desenvolver uma interface para a sua aplicação, colocando todas as funcionalidades que você fez no Python no back-end através de uma API. 

E terceiro: a performance é muito melhor. Ela é muito mais escalável e está pronta para lidar com centenas ou milhares de usuários simultâneos. 

Então, qual é o meu veredito? Eu acho que, se você quer levar a criação de agentes de IA a sério, eu recomendaria que você aprendesse os dois.
